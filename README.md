# Gastón Arballo - Concesionaria de Coches

Página web futurista y moderna para concesionaria de vehículos de alta gama.

## 🚀 Cómo subir a Internet

### Opción 1: GitHub Pages (Gratis)
1. Crea una cuenta en GitHub
2. Crea un nuevo repositorio
3. Sube los 3 archivos (index.html, styles.css, script.js)
4. Ve a Settings > Pages
5. Selecciona la rama "main" y guarda
6. Tu sitio estará disponible en: https://tunombre.github.io/nombre-repositorio

### Opción 2: Netlify (Gratis)
1. Ve a netlify.com
2. Arrastra la carpeta con los archivos
3. ¡Listo! Tendrás un URL inmediatamente

### Opción 3: Vercel (Gratis)
1. Crea cuenta en vercel.com
2. Importa tu proyecto desde GitHub o sube los archivos
3. Despliegue automático

### Opción 4: Hosting tradicional
1. Contrata un hosting (Hostinger, Bluehost, etc.)
2. Accede al cPanel o FileZilla
3. Sube los archivos a la carpeta public_html
